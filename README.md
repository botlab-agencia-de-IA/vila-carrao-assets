# vila-carrao-assets

Imagens públicas dos carrosséis do **Residencial Vila Carrão**.

Este repositório é **gerado automaticamente** pelo MazyOS. Cada pasta corresponde a um
carrossel, com os slides em PNG:

```
<nome-do-carrossel>/slide-01.png, slide-02.png, ...
```

## Por que existe

A API do Instagram e do Facebook publica imagens a partir de **URL pública** — não aceita
arquivo local nem repositório privado. O repositório principal do projeto é **privado**
(dados do negócio), então as fotos dos posts ficam aqui, num repo público separado contendo
**apenas PNGs** (nenhum dado sensível).

As fotos são servidas via:

```
https://raw.githubusercontent.com/botlab-agencia-de-IA/vila-carrao-assets/main/<carrossel>/slide-XX.png
```

## Não editar à mão

É saída automática. A fonte da verdade dos carrosséis vive no repositório privado
`botlab-agencia-de-IA/vila-carrao`.
